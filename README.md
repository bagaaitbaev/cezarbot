# CEZAR PS5 — Telegram бот (қазақ тілі)

Node.js + Telegraf + JSON (`data/store.json`). Брондау, комбо, қолмен уақыт, телефон `request_contact`, қиылысу тексеруі, 1 сағат бұрын ескерту.

## Қалай іске қосу (ноутбук, Windows)

1. [Node.js 18+](https://nodejs.org) орнатылған болсын.
2. Файл **`.env`** ішіне `@BotFather`-дан алған **`BOT_TOKEN=`** жазыңыз (жобада үлгі бар; токенді ешкімге көрсетпеңіз).
3. Папкада **`start.bat`** екі рет басыңыз: алдымен `npm install`, содан кейін ботты іске қосады. Терминалды **жабыңыз ба** — бот тоқтайды.
4. Немесе қолмен: `npm install` → `npm start`

`TZ=Asia/Almaty` — `.env` ішінде, қалаған болсаңыз өзгертіңіз.

**`TWO_GIS_REVIEW_URL`** (міндетті емес) — 2ГИС карточкасына тікелей сілтеме. Бос болса, хабарламалар жіберілмейді. Жаңадан жасалған броньдар үшін: визит аяқталғаннан **60 мин** өткен соң қонаққа екі тілде қысқа өтініш + **⭐ 2ГИС** батырмасы жіберіледі (ескі JSON-броньдарға тиіспейді).

### Операторлар

`.env`: **`OPERATOR_TELEGRAM_IDS`** — Telegram **user id** (немесе топтың id), бірнеше болса **үтірмен** (мысалы `123456789,-1001234567890`). Оператор алдымен ботқа **`/start`** жіберуі керек (жеке хабарлама алу үшін).

- Жаңа бронь **расталғанда** осы id-ларға хабарлама кетеді.
- **`/operator`** — оператор командаларының қысқаша нұсқауы (тек тізімдегі id үшін).
- **`/bugun`** немесе **`/today`** — бүгін басталатын расталған броньдар тізімі (тек оператор үшін).

### Сурет `/start` экранында

Папка **`assets/`**: файл **`welcome.jpg`**, **`welcome.png`** немесе **`welcome.webp`** (аты дәл `welcome`). Немесе `.env`: `WELCOME_IMAGE=assets/сіздің_файл.jpg`. Сурет жоқ болса — тек мәтін жіберіледі.

### Клиенттер мен броньдар бір таблицада (CSV)

Компьютерде папкада: **`npm run export:data`** → файл **`data/bookings_export.csv`** (Excel ашады): бронь, зона, уақыт, баға, `user_id`, телефон, аты. Бағандар **нүктелі үтір (;)** арқылы — орыс тілді Excelде екі рет басып ашқанда дұрыс бөлінеді.

## Қалай тексеру (локально)

1. **Автотесттер:** `npm test` — уақыт форматы, қиылысу, баға.
2. **Telegram:**
   - `/start` — сурет (болса) + 2 батырма: Брондау, Менің броньдарым.
   - **Брондау** → аймақ → уақыт (жылдам немесе «өзім енгіземін», мысалы `15:45`) → ұзақтық.
   - **1 сағат** — комбо сұрақсыз → телефон батырмасы → растау.
   - **3/5 сағат** — «Комбо пакет қосасыз ба?» → **Жоқ** немесе **Иә** → телефон → растау.
   - Телефонды тек **📱 Нөмірді жіберу** арқылы (қолмен емес) — растауға дейін.
   - **Өзгерту** — аймақтан қайта бастайды.
   - Бір уақытта бір аймақта **орындар санына** дейін бронь (Зал 5, Кабинка 3, ВИП 2); толық болса хабарлама.
   - **Менің броньдарым** → **Қайта брондау** — соңғы параметрлер + жаңа уақыт.

## Бағалар

`src/config.js` — `BASE_PRICES` (комбосыз) және `COMBO_PRICES` (комбо).

## Мәліметтер

`data/store.json` — броньдар мен пайдаланушылар (тек JSON, қосымша SQL пакеті жоқ).

## Егер сіз бағдарламашы болмасаңыз: ботты Telegram-да қалай іске қосу

Қысқаша **орысша** нұсқаулық:

### Бронь ережесі (бір аймақ, орындар)

**Зал** — бір уақытта ең көбі **5** бронь, **Кабинка** — **3**, **ВИП** — **2** (барлық уақыт белдеуі қиылысса, орын санын толтырады). Орындар толық болса, бот басқа уақытты ұсынады. `ZONE_CAPACITY` — `src/config.js`.

### Тегін сервер (ноутбуксыз 24/7)

Толық тегін және **ұзақ уақытқа** ыңғайлы нұсқа — **Oracle Cloud Free Tier** (Always Free): шағын VPS (немесе ARM), Node.js орнатып `npm start` немесе `pm2` арқылы фонда ұстау. Кредит карта керек (тексеру үшін), бірақ тегін лимит ішінде төлем жоқ.

Басқа нұсқалар (көбінесе **uyқы режимі** немесе лимиттер бар): **Fly.io**, **Render**, **Railway** — тегін тарифте бот **ұйықтап қалуы** мүмкін, сондықтан Telegram ботына сенімді емес.

**Қорытынды:** ноутбуксыз тұрақты жұмыс үшін **Oracle Cloud Free VM** немесе кез келген арзан VPS + `pm2` ұсынылады.

### Ботты Telegram-ға шығару (қадамдар)

1. **Telegram-да** `@BotFather` деп іздеңіз, чатты ашыңыз.
2. Жіберіңіз: `/newbot` — ботқа **аты** мен **username** беріңіз (username соңында `bot` болуы керек).
3. BotFather сізге **токен** жібереді (ұзын мәтін, мысалы `7123456789:AAH...`). Оны **ешкімге көрсетпеңіз** — бұл боттың кілті.
4. Компьютерде бот папкасында `.env` деген файл ашыңыз (немесе жасаңыз) және ішіне жазыңыз: `BOT_TOKEN=сіздің_токеніңіз`.
5. Терминалда (PowerShell) папкаға кіріп: `npm install` содан кейін `npm start`.
6. Бот **қосулы тұрғанда** ғана жауап береді. Телефонда Telegram ашып, өз ботыңызды **username** бойынша іздеңіз, **Start** басыңыз.

**Ескерту:** ботты үнемі жұмыс істету үшін ол күнде компьютер қосулы тұруы керек немесе серверге (VPS, хостинг) орнатылады — бұл бөлек тақырып, қажет болса айтыңыз.
